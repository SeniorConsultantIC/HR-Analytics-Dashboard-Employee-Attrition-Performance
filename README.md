This project analyzes employee attrition and performance using Power BI.

Key Features:
- Star Schema Data Model
- 5 Interactive Dashboard Pages
- 25+ DAX Measures
- HR KPIs: Attrition, Salary, Performance

Insights:
- High attrition in overtime employees
- Low satisfaction leads to higher exits
- Salary and tenure impact retention

Tools Used:
- Power BI
- DAX
- Data Modeling

Dataset:
Kaggle HR Analytics Dataset
